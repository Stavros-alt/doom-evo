# DOOM.EVO

A first-person shooter with evolving neural network enemies.

## Requirements

- Python 3.8+
- pygame
- numpy

## Installation

```bash
pip install pygame numpy
```

## Running

```bash
python main.py
```

## Controls

WASD: Move/Strafe, Mouse: Look, Left Click/Space: Shoot, P: Pause, ESC: Menu.

Enemies evolve via genetic algorithm each round.
